- O que são circuitos multiplicadores
- Projeto de multiplicador de 2 bits

### Pq não existe transporte entre o C0 -> C1 em um circuito multiplicador de 2bits?

Carry só pode surgir quando você soma pelo menos dois bits. Como:

0 + 0
0 + 1
1 + 0
1

nunca produz resultado maior que 1, não há transporte.
