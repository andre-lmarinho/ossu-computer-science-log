- Propriedades da função logica AND
  - A 0  = 0
  - A 1  = A
  - A A  = A
  - A A' = 0

- Propriedades da função logica OR
  - A + 0  = A
  - A + 1  = 1
  - A + A  = A
  - A + A' = 1

- Propriedades da função logica NOT
  - A + A'' = A

- Propriedades da função logica XOR
  - A + 1  = A'
  - A + 0  = A
  - A + A  = 0
  - A + A' = 1

- Propriedades Comutativa
  - A + B  = B + A
  - AB = BA

- Propriedades Associativas
  - A + (B + C) = (A + B) + C
  - A(BC) = (AB)C

- Propriedades Distributivas
  - A (B + C) = AB + AC
  - (A + B)(C + D) = AC + AD + BC + BD

- Propriedades de morgan
  - A' + B' = A'B'
  - A'B' = A' + B'

- Propriedade da cobertura
  - A + AB = A
